üîπ 2. Void Fn (Wid Args)

Yeh fn b kaafi simple hota h, lekin ye input leta h.
"Void" ka mtb h: kuch b return nhi krega, lekin args diye jayenge.
mtb, fn ko kuch values pass karni hoti h, pr vo values ko process karke result nhi dega ‚Äî bs apna kaam karega.

üí° Real-Life Analogy :
Sochiye ek chef (function) h jise biryani banane k liye bola gaya h.

üë®‚Äçüç≥ Chef ko ingredients (i/p) diye jate h ‚Äî masale, chawal, chicken, etc!
üí≠ Chef ingredients ko mix karega, pr final dish (o/p) aapko return nhi karega ‚Äî bs uska kaam complete hoga.

Yahi hota h void fn wid args:
  • üë®‚Äçüç≥ Chef = fn
  • üçö Ingredients = fn args (i/p)
  • ‚ùå Final dish = fn return value (nt returned)
  • ‚úÖ Biryani bn gyi (kaam complete!)

üîß Syntax (Structure)


void functionName(dataType arg1, dataType arg2) {
     // Yeh block un arguments ke saath kaam karega
}

Breakdown:

  • void: Fn kuch return nhi karega
  • functionName(): Fn ka naam + brackets m args (i/p)
  • { }: Fn ka kaam yha likha jata h, jo given args k saath perform hoga
Use Kab Karein?
Jab aapko kisi i/p ki zarurat ho, lekin aapko uska koi o/p ya result return nhi karna ho!

‚úÖ eg 1: Display a Custom Msg

    cout << "Hello, " << name << "!" << endl;
}

Explanation: Yeh fn ek naam lega aur usk liye personalized greeting dikhayega. Koi result return nhi karega, bs screen pe msg dikhayega!

‚úÖ eg 2: Print a Custom Line of Stars

     for (int i = 0; i < count; i++) {
        cout << "*";
     }
     cout << endl;
}

Explanation: Yeh fn num of stars print krega. i/p ke roop m aapko num of stars pass krna hoga!

‚úÖ eg 3: Display User Info

     cout << "Username: " << username << ", Age: " << age << endl;
}

Explanation: Ye fn user ka naam aur age dikhayega. Fn ko yeh information args ke roop m milegi.

Explanation: Fn call karte wqt args b pass kiye jaate h jo fn ke andar use hote h.
